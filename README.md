# Playwright C#

Playwright Learning Project (C#)
This repository is dedicated to exploring and mastering Playwright for browser automation and end-to-end testing using C#/.NET. The project serves as a personal learning environment where different scenarios, test cases, and best practices will be implemented step by step.

🚀 Roadmap
- Basic Playwright setup and first test
- Page Object Model implementation
- Advanced scenarios (file uploads, downloads, API mocking)
- Cross-browser and mobile emulation tests
- CI/CD integration and reporting

🛠️ Tech Stack
- Language: C# (.NET 8)
- Framework: Playwright for .NET
- Test Runner: NUnit
